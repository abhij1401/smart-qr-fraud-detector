
# Smart QR Code Fraud Detector

## Problem
Fraudsters place fake QR codes in public places and redirect users to phishing websites.

## Solution
A machine learning-powered system that analyzes URLs extracted from QR codes and predicts whether they are legitimate or malicious.

## Features
- URL feature extraction
- Phishing detection ML model
- Domain age lookup
- WHOIS integration
- VirusTotal verification
- Risk score generation
- FastAPI backend
- React QR Scanner frontend
